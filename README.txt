短视频素材收集分析平台 V6.8 社媒助手数据上报版

运行方法：
1. 双击 一键启动.bat
2. 打开 http://127.0.0.1:8000
3. 不要关闭黑色命令行窗口

新增：社媒助手数据上报对接
- 上报接口：http://127.0.0.1:8000/reporting
- 请求方法：POST
- 请求头：Content-Type: application/json; charset=utf-8
- 也支持：/api/import/social-assistant 和 /api/import/social_assistant

社媒助手建议勾选字段：
视频/笔记链接、标题、内容/文案、封面图/封面图链接、作者/博主昵称、作者主页/博主主页、发布时间、点赞量、评论量、收藏量、转发量、播放量、话题标签、视频播放链接/下载链接、数据更新时间。

说明：
社媒助手上报成功后，素材会自动进入“素材库”，并保留原有 AI 总结、AI 爆款分析、AI 文案改写、作者库、选题库等功能。
